Features

Full Stack Cinema Platform.

Built for Neighbourhoods

Everything you need to host or attend home theatre screenings.
From venue setup to ticket booking to verified reviews.

Venue Management

Create and manage home theatre venues with address, GPS coordinates, and screen type selection between 4K TV and Projector.

Interactive Seat Canvas

Configure seat layouts on a grid based canvas with 7 seat types: Chair, Single Sofa, Recliner, Three Seater Sofa, Single Bed, Double Bed, Triple Bed.

Show Scheduling

Schedule screenings with TMDB movie integration, set start and end times, pricing modes (free or paid), and privacy settings for invite only events.

Geolocation Map

Leaflet.js powered map with clustered markers for live shows, upcoming screenings, and venues. Distance based sorting for nearby discovery.

Seat Booking Engine

Interactive grid based seat selection with real time availability, type labels, capacity info, and price display. Book specific seats instantly.

Verified Reviews

Only ticket holders who attended a screening can leave reviews. 1 to 5 star ratings with comments. Build trusted local entertainment networks.

Dual Role System

Users select Host or Guest role after signup. Role specific navigation, permissions, and dashboards. Hosts manage venues, guests discover shows.

TMDB Movie Integration

Movie search and metadata powered by The Movie Database API with DNS over HTTPS resilience. Rich movie info for every screening.

Snack Menu

Hosts list available snacks with names, descriptions, pricing, and availability. Guests see what treats await at each screening.

Movie Requests

Guests request specific movies from hosts in their area. Hosts receive and respond to requests. Community driven screening schedules.

Auto Status Transitions

BullMQ job queue automatically transitions shows from Scheduled to Now Playing to Completed. Zero manual intervention required.

Private Shows

Password protected screenings for invite only events. Hosts set access codes, guests enter passwords. Perfect for exclusive movie nights.

Seat Engine

Interactive Seat Canvas.

Grid based seat configuration with 7 seat types. Hosts drag and place seats on a canvas. The layout generates a booking grid for guests with real time availability tracking.

Discovery

Geolocation Powered.

GPS based show discovery with distance sorting. Leaflet.js map with clustered markers for now playing and coming soon screenings. Find movie nights within walking distance.

Ticketing

Robust Booking Engine.

Select specific seats on the interactive grid. Free or paid modes set by hosts. Ticket lifecycle from booked to checked in to completed. Cancellation support for guests.

Automation

BullMQ Job Queue.

Redis backed job queue automatically transitions show statuses. Scheduled to Now Playing when start time hits. Now Playing to Completed when the show ends. Zero manual work.

Trust

Verified Review System.

Only attendees with completed tickets can review hosts. 1 to 5 star ratings with comments. Host profiles display aggregate ratings and review counts. Build community trust.

Security

Enterprise Security.

JWT authentication with bcrypt hashing. 3 tier rate limiting via Throttler. Helmet headers, CORS configuration, and class validator DTOs. Private shows with password protection.

FAQ

Frequently asked questions

Everything you need to know about hosting and attending home theatre screenings.

PartyOS is an open source platform that transforms home theatres into neighbourhood micro cinemas. Hosts set up their living rooms as screening venues, configure seating layouts, schedule movie shows, and sell or offer free tickets. Guests discover nearby screenings, book seats, and review hosts.
Yes. AGPL 3.0 licensed, no usage limits, no restrictions on the number of venues, shows, or users. You self host it on your own server and you own everything. The only cost is your server and infrastructure.
Backend: NestJS 11 with Fastify adapter, Prisma ORM, PostgreSQL 16, Redis 7, and BullMQ for job queues. Frontend: React Native with Expo SDK 55, TypeScript, Leaflet.js maps. Everything containerized with Docker Compose for one command deployment.
Hosts configure their seating layout on an interactive canvas with 7 seat types (chairs, sofas, recliners, beds). This generates a booking grid. Guests see real time seat availability, select specific seats, and book instantly. Tickets track status from booked to checked in to completed.
Only guests who have purchased a ticket and attended a screening (ticket status completed) can leave a review. This prevents fake reviews and builds genuine trust in the community. Reviews include 1 to 5 star ratings with comments.
Yes. Hosts can set a password on any show to make it private. Guest must enter the correct password before booking. Perfect for exclusive movie nights with friends and family, or community events with controlled access.
Any server that can run Docker. A basic VPS with 2GB RAM is sufficient to get started. The Docker Compose setup includes the NestJS backend, PostgreSQL database, and Redis cache. No GPU required. Works on any Linux, macOS, or Windows server with Docker installed.
